# Design System: MANAGYM Premium Modern
**Project ID:** 2479863847744860202

## 1. Visual Theme & Atmosphere
"Boutique Fitness Luxury" — A sophisticated, high-end evolution of the neon theme. It uses deeper color layers, smoother transitions, and "Super-ellipses" (continuous curves) for a more organic yet hi-tech feel. Atmosphere is "Exclusive," "Precise," and "Premium."

## 2. Color Palette & Roles
* **Void Navy (#05070a):** The deep, velvet-black background.
* **Cyber Emerald (#10b981):** A more refined, slightly leaf-tinted neon green for success and activity.
* **Electric Indigo (#6366f1):** Replaces the basic blue for a more vibrant, luxurious feel.
* **Solar Flare (#f59e0b):** A sophisticated amber for highlighting coaching and premium services.
* **Boutique Glass (rgba(15, 23, 42, 0.4)):** Darker, more tinted glass with heavy backdrop-blur (24px).
* **Silver Stroke (rgba(255, 255, 255, 0.05)):** Ultra-thin borders for a precise "etched" look.

## 3. Typography Rules
* **Font Family:** 'Plus Jakarta Sans' or 'Outfit' (fallback to 'Inter').
* **Headers:** Large, bold, with slightly negative letter-spacing (-0.02em).
* **Body:** Clean sans-serif with increased line-height (1.6) for an airy feel.

## 4. Component Stylings
* **Buttons:** "Liquid" feel. Rounded (12px), subtle inner-shadow, and a hover "shimmer" effect. 
* **Cards/Containers:** Generous padding (32px), "Super-ellipse" corners (24px), invisible borders except for a top "glass edge" highlight.
* **Navigation:** Minimalist, using icons with subtle "Glow-on-Active" states.

## 5. Layout Principles
* **Padding:** Strict 8px grid (gaps of 16px, 24px, 32px).
* **Hierarchy:** High contrast between primary surfaces and the "Void" background.
* **Density:** Spacious (Low density) for a luxury experience.
